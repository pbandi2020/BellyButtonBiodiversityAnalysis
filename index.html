Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

Roza wanted to utilize her Belly Button data samples collected frpm volunteers and program a functioning website where users can vizualize the data and looks up their unique data to find the probable match for the bacteria used in the Improbable Beef Burger.

The goal of the project was plotting belly button bacteria samples and creating a live dashboard accessed by user input.

1. Bar Chart - Creates a horizontal bar chart to with interactive dropdown menu to change the value and select the top 10 OTU individuals found
2. Gauge Chart - Shows us the bacteria culture sample
3. Bubble Chart - Displays each sample metadata collected
